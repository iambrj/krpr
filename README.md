# krpr

Creeper is a library for commonly used relations in miniKanren

## Usage

- Add an appropriate version of miniKanren to the root directory and `require`
    any relations you need (replace `"../faster-mk/mk.rkt"` with corresponding
    module)

## Tests

- To run all tests, do

    ```
    raco test ./tests/tests.rkt
    ```
